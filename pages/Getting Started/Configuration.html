Configuration

Hindenburg has an easy-to-use JSON file for configuring the whole server. Hindenburg will look for a config.json in the current working directory, or if the HINDENBURG_CONFIGS environment variable is set to an absolute filename of the config.json to use, check out the Environment Variables page for more information.

Configuration Values

config.versions

An array of game versions that Hindenburg will accept.

Type: array

Default: [ "2021.11.9" ]

config.versions[]

Type: string

config.clusterName

The name of the cluster that this node belongs to.

Type: string

Default: "Capybara"

The ID of this node in relation to other nodes in the cluster.
